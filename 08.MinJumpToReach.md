# Minimum Jumps

```java
class Solution {
    static int minJumps(int[] arr) {
        int dp[] = new int[arr.length];
        dp[arr.length - 1] = 0; // No jumps needed from the last position

        for(int i = arr.length - 2; i >= 0; i--) {
            if(arr[i] == 0) {
                dp[i] = Integer.MAX_VALUE; // No jumps possible from a 0
            } else {
                int step = Integer.MAX_VALUE;
                for(int k = 1; k <= arr[i] && i + k < arr.length; k++) {
                    if(dp[i + k] != Integer.MAX_VALUE) { // Only consider valid steps
                        step = Math.min(step, dp[i + k]);
                    }
                }
                if(step != Integer.MAX_VALUE) {
                    dp[i] = step + 1; // Update the dp with the minimum step count
                } else {
                    dp[i] = Integer.MAX_VALUE; // No valid jumps from this position
                }
            }
        }
        return dp[0] == Integer.MAX_VALUE ? -1 : dp[0]; // Return -1 if no path is found
    }
}
```
